article.css download page

Download

Basic responsive style for simple pages.

Compressed version: article.css

Development version: article.min.css

Last version: 1.0

Properties
  1. No CSS classes, only new and modified HTML tags.
  2. No JS.
  3. Responsive design.

HTML tags

Modified tags

Tags: body, a, a:hover, a:active, a:visited, section, article, p, h1, h2, h3, h4, h5, h6, img, pre, code

New tags
  1. block - Instead of the div tag.
  2. aleft, aright, acenter, ajustify - Instead of the left, right, center, justify tags and CSS formats.
  3. font-helvetica - CSS: font-family: Helvetica, Arial, sans-serif;
  4. font-code - CSS: font-family: "Lucida Console", Monaco, monospace;
  5. font-palatino - CSS: font-family: "Palatino Linotype", "Book Antiqua", Palatino, serif;
  6. font-roman - CSS: font-family: "Times New Roman", Times, serif;
  7. thumbnail-500 - For images display: inline-block; max-width: 500px; height: auto; Place before and after the img tag!
  8. thumbnail-170 - For images display: inline-block; max-width: 170px; height: auto; Place before and after the img tag!
