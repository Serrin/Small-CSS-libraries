article.css download page

Download

Basic responsive style for simple pages.

Compressed version: article.css

Development version: article.min.css

Last version: 1.0.1

Properties
  1. No CSS classes, only custom and modified HTML tags and tag attributes.
  2. No JS.
  3. Responsive design.

HTML tags

Modified tags

Tags: body, a, a:hover, a:active, a:visited, section, article, p, h1, h2, h3, h4, h5, h6, img, pre, code

New tags
  1. block - Instead of the div tag.
  2. aleft, aright, acenter, ajustify - Instead of the left, right, center, justify tags and CSS formats.
  3. font-helvetica - CSS: font-family: Helvetica, Arial, sans-serif;
  4. font-code - CSS: font-family: "Lucida Console", Monaco, monospace;
  5. font-palatino - CSS: font-family: "Palatino Linotype", "Book Antiqua", Palatino, serif;
  6. font-roman - CSS: font-family: "Times New Roman", Times, serif;
  7. thumbnail-500 - For images display: inline-block; max-width: 500px; height: auto; Place before and after the img tag!
  8. thumbnail-170 - For images display: inline-block; max-width: 170px; height: auto; Place before and after the img tag!

Basic grid

 
<!DOCTYPE html>
<html lang="en">
<head>
<!--[if lt IE 9]>
<script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.js"></script>
<![endif]-->
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
<title>page title</title>
<link href="article.min.css" rel="stylesheet" type="text/css">
</head>
<body>
<ajustify><font-helvetica>
	<section>
		<article>
			<acenter><h1>main title</h1><acenter>
			<block>
				<acenter><h2>subtitle</h2><acenter>
				<p>some text</p>
			</block>
		</article>
	</section>
</font-helvetica></ajustify>
</body>
</html>

Colors

Formats
  1. bg-colorname - Background color with black or white text color.
  2. color-colorname - Text color.

Samples

A block with fuchsia background and white text color.

A block with green text color.

Combined: A block with navy background and yellow text color.

Sample code:

<block bg-fuchsia><aleft><p>A block with fuchsia background and white text color.</p></aleft></block>
<block color-green><aleft><p>A block with green text color.</p></aleft></block>
<block bg-red color-yellow><aleft><p>Combined: A block with red background and yellow text color.</p></aleft></block>
